# 🔧 Variables de Entorno para Evolution API v2.2.3

## 📋 **VARIABLES PRINCIPALES**

```bash
# Autenticación
AUTHENTICATION_API_KEY=clickup-whatsapp-v223

# Base de datos
DATABASE_PROVIDER=postgresql
DATABASE_URL=${{PostgreSQL.DATABASE_URL}}

# QR Code (configuración mejorada)
QRCODE_LIMIT=30
QRCODE_COLOR=#198754

# Configuración de sesión
CONFIG_SESSION_PHONE_CLIENT=Evolution API
CONFIG_SESSION_PHONE_NAME=Chrome

# Webhook (opcional, para después)
WEBHOOK_ENABLED=false

# Logs
LOG_LEVEL=ERROR
LOG_COLOR=true

# Configuración del servidor
SERVER_PORT=8080
SERVER_URL=https://tu-nuevo-evolution.up.railway.app

# Cache y Redis
CACHE_ENABLED=true
CACHE_TTL=3600

# Configuración de instancias
DEL_INSTANCE=false
DEL_TEMP_INSTANCES=true

# Configuración de grupos
GROUPS_IGNORE=true

# Configuración de eventos
WEBHOOK_EVENTS_MESSAGES_UPSERT=false
WEBHOOK_EVENTS_MESSAGES_UPDATE=false
WEBHOOK_EVENTS_MESSAGES_DELETE=false
WEBHOOK_EVENTS_SEND_MESSAGE=false
WEBHOOK_EVENTS_CONNECTIONS_UPDATE=false
WEBHOOK_EVENTS_PRESENCE_UPDATE=false
WEBHOOK_EVENTS_CHATS_UPSERT=false
WEBHOOK_EVENTS_CHATS_UPDATE=false
WEBHOOK_EVENTS_CHATS_DELETE=false
WEBHOOK_EVENTS_CONTACTS_UPSERT=false
WEBHOOK_EVENTS_CONTACTS_UPDATE=false
```
